Detailed Description

This class deals with all the sensors' information and the fused result.

SensorInfor class

fusion function:
sigma(SV, X) = p1 * D + p2
sigma(RADAR, X) = u1 * exp(u2 * D)
sigma(SV, Z) = u3 * exp(u4 * D)
sigma(RADAR,Z) = p3 * D + p4
